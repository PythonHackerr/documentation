Documentation

Radar is location infrastructure for every product and service. You can use our SDKs and APIs to build a wide range of use cases, including pickup and delivery tracking, location-based messaging, location verification, store locators, address autocomplete, and more.

Products#

Geofences

Industry-leading accuracy with unlimited geofences, polygon geofences, and more

Trip tracking

Trip tracking, live ETAs, arrival detection, and routing for pickups and deliveries

Places

Points-of-interest (POI) dataset to detect visits to millions of places

Regions

Admin boundary dataset for country, state, and postal code detection

Beacons

Hardware-enabled micro-geofences accurately detect location down to a meter or less

Fraud

Detect GPS spoofing, proxy and VPN usage, and device tampering

Geocoding APIs

Forward, reverse, and IP geocoding APIs

Search APIs

Autocomplete and place search APIs

Distance APIs

Distance and matrix APIs to calculate travel distance and duration
